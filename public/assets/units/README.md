# Unit Assets

วางไฟล์ภาพ PNG (พื้นหลังโปร่งใส) ของแต่ละ Unit ไว้ในโฟลเดอร์ย่อยตามชื่อ Unit

## โครงสร้างไฟล์ที่ต้องการ (ต่อ Unit 1 ตัว)
```
triangle/
├── icon.png          ← ไอคอนสี่เหลี่ยม (128x128) สำหรับ Unit Panel
├── portrait.png      ← ภาพ Full Art สำหรับ Encyclopedia/Info
├── chibi.png         ← ตัว Chibi (128x128) สำหรับวาดบน Canvas
├── face_right.png    ← หันขวา (64x64)
├── face_left.png     ← หันซ้าย (64x64)
├── face_front.png    ← หันหน้า (64x64)
├── face_back.png     ← หันหลัง (64x64)
└── skill_icon.png    ← ไอคอนสกิล Tier 3 (64x64)
```

## ขนาดภาพแนะนำ
- icon.png: 128x128 px
- portrait.png: 512x512 px หรือ 512x720 px
- chibi.png: 128x128 px
- face_*.png: 64x64 px
- skill_icon.png: 64x64 px
